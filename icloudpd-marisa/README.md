# [iCloud PD](https://github.com/boredazfcuk/docker-icloudpd) Marisa

Automatically download photos and videos saved in iCloud.

Rename the file [`icloudpd_templ.conf`](/icloudpd-marisa/icloudpd_templ.conf) to `icloudpd.conf` and set/change the values of the following fields in the config:
- `apple_id`
- `group`
- `group_id`
- `user`
- `user_id`
For Telegram notifications:
- `telegram_chat_id` 
- `telegram_token`
