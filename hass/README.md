# Home Assistant

This configuration uses the UI Lovelace Minimalist package for the styling of the dashboard. It includes cards for the following:
- Publish the Reolink cameras in Frigate with `two-way-audio`
- Turn Windows PCs Off and On through `Wake-On-Lan and netRPC shutdown`
- Different cards `based on HA-user`
- `Responsive design for phones` with pop-ups for each defined room
- Climate control with Homematic IP
- Show gas prices, the trash calendar and the weather
- Buttons to open garages and locks
- Light and TV controls
- QR-Code for quick pair to WiFi
